# smart-Quiz
A simple and user-friendly quiz web application that allows users to select multiple subjects, choose difficulty levels, and attempt quizzes within a timer. The project is built using HTML, CSS, JavaScript, Node.js, and Express.js, with a clear and structured data flow between the frontend, backend, and database through APIs, making it easy to understand and use. It also includes an Analytics tab where users can track their progress such as accuracy, average performance, and total quizzes attempted, along with a Profile tab where users can view their profile details.
